# middleman-slim

middleman-slim is an extension for the Middleman static site generator that adds support for Slim.

## Installation

    gem install middleman 
    gem install middleman-slim 
    middleman init PROJECT_NAME --template slim

## Run 
 
    cd PROJECT_NAME
    middleman server
